Dev Infos

to start the python programm correctly, run:
->  start-app.bat

to save requirements to file:
->  pip freeze > requirements.txt

to reinstall requirements:
->  pip install -r requirements.txt

to package the app:
->  pyinstaller --onefile --icon=Dara-3.ico Daras_hotkey_listener.py

if not installed:
->  pip install pyinstaller
